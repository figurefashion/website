# MIRROR

**Real-time fit infrastructure for fashion commerce.**

MIRROR gives fashion brands tailor-level fit accuracy online - using a single camera and real body geometry, in real time.

## About

MIRROR replaces guesswork with geometry. One camera. Real body data. Your garments fitted to actual shape.

- **Problem**: Over 70% of returns are driven by sizing. Customers guess because they never actually see fit.
- **Solution**: Camera → 3D Body Geometry → Simulation → Size Decision

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: TailwindCSS
- **Typography**: Geist Sans
- **Form Handling**: Formspree

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
src/
  app/
    page.tsx      # Main landing page
    layout.tsx    # Root layout with metadata
    globals.css   # Global styles and Tailwind imports
```

## Deployment

Deploy on Vercel or Netlify. The site is static-ready.
